## ğŸ“¦ Order Management

### Receiving Orders

When a new order arrives:
1. **Notification** - Sound and visual alert
2. **Review** - Check order details and items
3. **Accept/Reject** - Confirm within 2 minutes
4. **Prepare** - Get items ready for pickup
5. **Ready** - Mark order ready for rider pickup

### Order Workflow

```
New Order â†’ Accept â†’ Prepare â†’ Ready for Pickup â†’ Rider Pickup â†’ Delivered
```

### Managing Orders
- **Accept quickly** - Fast acceptance improves ratings
- **Communicate** - Contact customer if items unavailable
- **Accurate timing** - Provide realistic preparation time
- **Mark ready** - Alert rider when order is ready
- **Quality check** - Ensure proper packaging

### Handling Issues
- **Item unavailable**: Offer substitutes or refund
- **Delay**: Notify customer through app
- **Incorrect order**: Fix before rider pickup
- **Customer changes**: Update order if not yet prepared
